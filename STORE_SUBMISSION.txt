── パッケージの概要（短文・132文字以内／manifest の description と同じでも可） ─
Cleaner line breaks for Japanese captions on YouTube—powered by Budoux.


── 説明（最大 16,000 文字用。英語で書いてよい。そのまま下のブロックを貼る） ─────
Better Japanese Captions reflows Japanese subtitle lines on YouTube so they break at more natural places—easier to read at a glance.

How it works:
• Runs only on youtube.com pages (desktop and mobile domains declared in the manifest).
• Uses Budoux-style segmentation bundled inside the extension—no caption text is sent to a developer-operated server.
• Includes an on-player toggle to restore YouTube’s original caption layout when you want to compare.
• Optional keyboard shortcut (default Alt+Shift+C) toggles the same on/off state; you can change it under chrome://extensions/shortcuts.

Privacy:
• No accounts, no analytics SDKs, no remote caption uploads by this extension.
• See the Privacy Policy linked from the listing for details.

YouTube の日本語字幕が、読みやすい位置で改行されるようにレイアウトを調整します。処理はすべてブラウザ内で行われます。

• 対象サイトはマニフェストで指定した YouTube（youtube.com／www／m のみ）。
• Budoux に相当するモデルを同梱しローカルで分割。開発者運用サーバーへの字幕送信はありません。
• プレーヤー横のトグルで、本来の字幕レイアウトに戻して比較できます。
• ショートカットキー（既定 Alt+Shift+C）でも同じオン・オフ切り替えが可能。割り当ては chrome://extensions/shortcuts で変更できます。
• アカウント不要・アプリ内解析・字幕データの開発者側収集なし。

Single purpose（単一用途）、英語:
This extension has one purpose: to improve readability of Japanese captions on YouTube by adjusting line breaks locally in the browser.


Host / site access justification（英語）:
Access is limited to YouTube domains where captions appear (youtube.com, www.youtube.com, m.youtube.com). The content script reads and adjusts only the caption DOM on those pages.


Manifest（権限の事実確認）:
No host_permissions wildcard. Host access only via content_scripts.matches for https://youtube.com/, https://www.youtube.com/, https://m.youtube.com/ . Additional permission: activeTab (see justification below). No tabs-wide permission, no storage permission in manifest.


── activeTab を要求する理由（プライバシーへの取り組み／日本語・ストア貼り付け用） ──
ユーザーがブラウザのショートカット（既定: Alt+Shift+C）を押したときに、今表示している YouTube のタブ内のコンテンツスクリプトへ「字幕の行頭改行オンのオン／オフ」を伝えるためです。フォーカス中のタブへメッセージを送る処理は、ユーザーが明示的にショートカットを操作したときに限り有効になるため、activeTab を使います。常時すべてのタブを横断して読み取りは行わず、ショートカット未使用時は追加のタブ権限は付与されません。対象はマニフェストで限定した YouTube（youtube.com／www／m）上のプレーヤー設定の切り替えのみです。


activeTab justification (English, for reference / alternate locale):
The extension registers a keyboard command so users can toggle caption line-break reflow without opening the player controls. When the user presses the shortcut, the service worker needs to send a one-shot message to the content script in the active tab. The activeTab permission grants that access only in response to this explicit user gesture (the shortcut), not for passive or background tab access. We do not request broad tabs permission; we do not read unrelated sites.


データ開示・アンケート用の要点:
• Developer does not remotely collect caption text or sell user data.
• Processing is local; toggle state may persist on the YouTube origin only via normal browser storage.
